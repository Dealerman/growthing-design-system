# Growthing — Website UI Kit

A faithful recreation of what a Growthing marketing site would look like, built from the visual language in the proposal deck. There is no public source codebase for growthing.info accessible to us, so these components are an extrapolation of the deck's brand system — not a screenshot-perfect clone.

## Files
- `index.html` — full homepage prototype (hero → services → methodology → team → pricing → contact)
- `components.jsx` — reusable components (Header, Hero, ServicesGrid, Flowchart, TeamBlock, PricingCard, ContactSection, Footer)

## Approach
- Spanish-first copy, lifted directly from the proposal deck.
- Magenta-on-white with dark sections for contrast.
- Round magenta service tiles, zero-prefixed step numbers, pill contact capsules — all directly from the source deck.
- Lucide icons substitute for the bespoke deck icons (see README · ICONOGRAPHY).

## Caveats
This kit is a brand extrapolation, not a 1:1 website recreation. If Growthing provides their live site's HTML/CSS or a Figma file, re-open this kit and I'll cross-reference and correct.
